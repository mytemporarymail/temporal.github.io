    对师，该如何供养？

    （本文摘自惟海法师2007年于新加坡所讲法华经的录音文字整理）

    我们作为佛子，佛的弟子，我们按照佛陀的教导，实实在在地修行，这就是供养，是对佛的供养。

    以前我看到经典当中，对佛希望勉励弟子修行这一点我不是特别看重，尽管我已经当教师了。后来当教师当久了，时间长了，慢慢琢磨出味道了。

    特别是人进入中年以后，就体会到一个东西，我们对学生的教育，其实我们没有目的要求，例如希望学生招待我什么，学生给我个礼品，学生能供养我个大红包。有的时候，你不高兴，觉得很生气，我教的东西你不学，你拿这些东西，对我究竟有什么见识？

    所以一个好的学生，应该把老师教的东西统统学到手，老师没有教过的东西他能弥补起来，老师没有发扬光大的东西发扬起来。

    古代祖师讲过一句话，在《五灯会元》上看到的，说“弟子与师同，减师半德；弟子不如师，断吾子孙。”什么意思？弟子如果水平跟老师差不多，那么老师的一半功德没了；如果弟子比老师还好，那么老师也引以为自豪，他一辈子都会记得；如果那个徒弟比老师还差，一代比一代差，到第四代，可能就绝种了，灭亡了，什么东西都传不下去了。

    我在读书的时候，很多老师都特别喜欢我，其中有两个老师把我抓得特别厉害，都想抓我读他们的研究生，或者留校。我工作以后，他们经常给我写信。我出家后，他们还是经常给我写信。

    前两年，我那个老师来看我，在路上心肌梗塞了，这是第二位因为我而心肌梗死的大学老师，他的大女儿来告诉我的，我心情很沉重。

    我的书一出来，就立即寄给他，他是最早看我的书，而且是认认真真看完的。他立即给我写了一封信，其中给我说了一句“弟子的成就已经超过了老师，你虽然现在不搞我的专业。”

    我回去看他，他特别高兴，我们师母也知道，他的儿女也知道。

    所以老师去世以后，他们家里和同学，我只是一个本科生，不是他的硕士博士，他的硕士博士学生很多，国外的也有，包括黑种人。但是他特别记住我，所以老师去世后，家里的儿女就通知我，我就回去了，我觉得这样给老人是一种安慰。

    我这是想这个道理，因为对一个好的弟子——我不是说我有多好，一个好的弟子，能够使老师的学术生命发扬光大。

    释迦牟尼佛开辟了佛法，创立了佛教，肯定是希望我们弟子在佛法上发扬光大，这样才能够为佛增光，把佛法传扬出去。如果我们一代不如不代，最后佛法就丧失在我们手里，这不是断佛慧命么？

    慧思大师曾经给智者大师讲过一句话。因为我们是讲《法华经》，讲天台，所以我讲天台宗的故事，天台宗的典故。

    当年智者大师在慧思大师那学成之后，要开辟弘法事业的时候，慧思大师说：“你出去，不要作佛法断种人。”如果他不如慧思大师，最后恐怕一代都传不下去。

    而事实上，智者大师超过了慧思大师，正式建立了中国第一个佛教宗派，建立起了庞大的天台宗佛法体系。他真正把佛法传了下去，没有做了断种人。

    我们作为佛子，学习佛法，我们今天学习《法华经》，作为观世音菩萨的信仰者，学习观世音菩萨，那么我们要怎么供养观世音菩萨呢？我觉得这个最重要。

    所以说，我们去供养老师的时候并不是稀罕我们的东西，给一个红包。我们如果能拿得出一个业绩，他看到他的学生那么有出息那么有成绩，那对他的价值——精神价值、生命价值，他会觉得他的生命延续了，扩大了，在人间流行，他会觉得他的生命永生。

    这个价值多大！这个供养多大！什么钞票，什么点心，比得上这个供养？
